Change password

Passwords are a critical component of information security and serve to your account.

  • The password should not be same as user name or as last used password
  • The password should be at least eight characters long.
  • The password should contain characters from three of the following four categories:
  • Latin uppercase & lowercase letters
  • Base 10 digits (0 through 9)
  • Non-alphanumeric characters
  • Passwords can be up to 20 characters long
