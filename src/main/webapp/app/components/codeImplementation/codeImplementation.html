CODE AND IMPLEMENTATION

What does the code do?
The code enables websites to ask visitors for permission to send push notifications.
You can choose to add code only on the page(s) on which you want permission box-shadowfor notifications to appear.
The permission box to ask for permission will not appear unless you add this code on your website page(s).

Copy and Paste this code before {{head}} tag on your website page(s)

If you are using HTTPS certificate, please download this service worker file and place it in the root folder of your site.
